Модуль 1: Базовая настройка устройств и сетевых сервисов

Данный модуль охватывает настройку имен хостов, IP-адресации, подсетей, NAT, DHCP, OSPF, GRE и сервисов на Linux-серверах, включая DNS и SSH. Конфигурации выполняются как на сетевом оборудовании, так и на виртуальных машинах.

Конфигурация устройств

# Вход в режим глобальной конфигурации
configure terminal
hostname isp.au-team.irpo
exit
write memory

configure terminal
hostname hq-rtr.au-team.irpo
exit
write memory

configure terminal
hostname br-rtr.au-team.irpo
exit
write memory

Настройка имени хоста в Linux

sudo hostnamectl set-hostname hq-srv.au-team.irpo
sudo bash -c 'echo hq-srv.au-team.irpo > /etc/hostname'
...

Подсети и IP-адреса

  • VLAN 100 (HQ-SRV): 192.168.10.0/26
  • VLAN 200 (HQ-CLI): 192.168.10.64/28
  • VLAN 999 (Управление): 192.168.10.80/29
  • BR-SRV: 192.168.20.0/27
  • ISP к HQ-RTR: 172.16.4.0/28
  • ISP к BR-RTR: 172.16.5.0/28

Настройки интерфейсов и IP

interface vlan 100
 ip address 192.168.10.1 255.255.255.192
 no shutdown
...

Netplan конфигурации

network:
  version: 2
  ethernets:
    eth0:
      addresses:
        - 192.168.10.2/26
      gateway4: 192.168.10.1
      nameservers:
        addresses: [192.168.10.2]

Настройка NAT, DHCP, GRE, OSPF

ip nat pool HQ_POOL 172.16.4.1 172.16.4.1 netmask 255.255.255.240
ip nat inside source list 1 pool HQ_POOL overload
...
router ospf 1
 network 10.0.0.0 0.0.0.3 area 0
...

DNS-сервер (BIND)

zone "au-team.irpo" {
  type master;
  file "/etc/bind/db.au-team.irpo";
};

SSH и безопасность

Port 2024
PermitRootLogin no
AllowUsers sshuser
...
